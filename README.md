Cyrax is microframework to build layered architecture, one of the core concepts of Domain Driven Design

## Installation

Add this line to your Gemfile.

Rails 4:

    gem 'cyrax', '~> 0.4.0'

Rails 3:

    gem 'cyrax', '~> 0.3.8'

## Usage

Look at `example` folder.
